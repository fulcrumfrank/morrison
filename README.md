# ADU Project Site

A Next.js starter repo for a public-facing ADU project website and interactive pro forma tool.

## What is included

- Multi-page site structure for the ADU project
- Placeholder content pages for Vision, Site, Design, Process, and Documents
- Interactive Financial Model page
- Adjustable assumptions
- 5 / 10 / 15 year views
- Monthly / quarterly / annual aggregation
- Charts and CSV export

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Recharts
- Lucide React

## Getting started

```bash
npm install
npm run dev
```
